# EscapeFromDeath — исправленная базовая версия

Я подготовил проект так, чтобы он открывался и собирался в Visual Studio.

## Что было в архиве

В исходном архиве было две похожие структуры проекта:

- `EscapeFromDeath.sln` -> использует проект `EscapeFromDeath/EscapeFromDeath.vcxproj`
- `EscapeFromDeath/EscapeFromDeath.sln` -> использует проект `EscapeFromDeath/EscapeFromDeath/EscapeFromDeath.vcxproj`

Чтобы не гадать, какой `.sln` вы будете открывать, базовая игра вставлена в обе папки проекта.

## Как запустить

1. Откройте `EscapeFromDeath.sln` в Visual Studio.
2. Выберите конфигурацию `Debug x64`.
3. Нажмите `Build -> Build Solution`.
4. Запустите проект.

Можно также открыть вложенный `EscapeFromDeath/EscapeFromDeath.sln` — он тоже должен собраться.

## Что реализовано

- главное меню;
- новая игра;
- загрузка и сохранение;
- 9 дней подготовки;
- финальный побег на 10-й день;
- 8 локаций тюрьмы;
- тренировки силы, ловкости и выносливости;
- душ для восстановления здоровья;
- инвентарь;
- NPC: Контрабандист, Библиотекарь, Силач, Игроман;
- базовая боевая система с быстрым вводом слова;
- внешние файлы ресурсов в папке `data`.

Все заголовочные файлы имеют расширение `.h`, не `.hpp`.
