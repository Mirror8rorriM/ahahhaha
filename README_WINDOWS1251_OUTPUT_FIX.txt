Исправление русского вывода без UTF-8

Что изменено:
1. Убран параметр компилятора /utf-8 из проекта Visual Studio.
2. Убран #pragma execution_character_set("utf-8") из .cpp и .h файлов.
3. Консоль Windows переключается на кодовую страницу 1251 через SetConsoleCP(1251) и SetConsoleOutputCP(1251).
4. Текстовые исходники и файлы data сохранены в Windows-1251.

Важно:
- В Visual Studio открывайте и сохраняйте эти файлы как Windows-1251 / Cyrillic.
- Если русский снова станет кракозябрами, проверьте, что файлы случайно не пересохранились в другой кодировке.
